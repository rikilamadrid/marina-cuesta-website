# marina-cuesta-website

Marina Cuesta's portfolio site — an editorial monograph built with Next.js (App Router), TypeScript, Tailwind v4, Sanity CMS, and Framer Motion, deployed on Vercel.

See [`CLAUDE.md`](./CLAUDE.md) for working conventions and [`context/`](./context/) for the full project spec, design reference, and feature specs.

## Development

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).
